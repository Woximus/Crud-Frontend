# 👥 System Zarządzania Kadrami (Frontend)

Aplikacja webowa w **Angularze** (Single Page Application) do zarządzania pracownikami. Projekt stawia na przejrzysty interfejs (Bootstrap 5) oraz bezpieczeństwo.

**Główne cechy:**
* Pełny CRUD pracowników (dodawanie, edycja, zwalnianie).
* Zabezpieczony routing (Auth Guards) – brak dostępu bez zalogowania.
* Dynamiczna walidacja formularzy (odrzucanie niepoprawnych danych na starcie).

---

## 🚀 Jak uruchomić projekt?

1. Upewnij się, że masz zainstalowanego **Node.js**.
2. Otwórz terminal w folderze projektu i pobierz wymagane pakiety:
   ```bash
   npm install
3. Uruchom serwer developerski:
   ```bash
   ng serve -o

Aplikacja automatycznie otworzy się w przeglądarce pod adresem http://localhost:4200.
